# No.1 Shoes Bolte | Bolte Glow Forge

Premium sneakers, sports shoes & streetwear in Bhiwandi, Thane.

## Project info

This is a premium footwear e-commerce application.

## How can I edit this code?

You can work locally using your own IDE, clone this repo and push changes.

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- Three.js (React Three Fiber)

## Deployment

You can deploy this project to platforms like Vercel, Netlify, or AWS by connecting your repository.
